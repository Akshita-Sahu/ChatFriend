# ChatFriend 

A powerful AI Chat Platform built with Java.
Supports multiple AI providers like OpenAI, Claude, Gemini, DeepSeek and more.

##  Features
- Multi AI Provider Support (OpenAI, Claude, Gemini, DeepSeek, Ollama, Azure)
- Enterprise Knowledge Base
- AI Chatbot Builder
- User Authentication & Role Management

##  Tech Stack
- Backend: Java (Spring Boot)
- Frontend: Vue + TypeScript
- Database: MySQL
